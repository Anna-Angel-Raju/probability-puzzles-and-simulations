#  Probability Puzzles and Simulations

##  Overview

This repository contains a collection of classic **probability puzzles**, explained with strong intuition and supported by **Python simulations**.

The goal is to bridge the gap between:

* Mathematical theory 
* Real-world intuition 
* Computational verification 

---

## 🧩 Problems Included

### 1. Monty Hall Problem

* Conditional probability paradox
* Demonstrates why switching improves winning chances
